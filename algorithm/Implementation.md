# Implementation

대부분의 문제는 implementation을 필요로 함.

흔히 알고리즘 대회에서 implementation 문제는?

- 풀이를 떠올리기는 쉽지만 소스코드로 옮기기 어려운 문제: 많은 연습이 필요함
  - 알고리즘은 간단한데 코드가 지나칠만큼 길어지는 문제 -> 언어마다 다름
  - 실수 연산을 다루면서 특정 소수점 자리까지 출력하는 문제
  - 문자열을 특정 기준에 따라 끊어 처리하는 문제
  - 적절한 라이브러리를 사용해야 하느 문제 -> ex. 순열, 조합 찾는 문제 (python itertools)

- 일반적으로 2차원 공간은 행렬(Matrix)의 의미로 사용됨
  - 행렬의 의미를 알아두어야됨

- 2차원 공간의 방향 벡터도 자주 사용됨

```
# x, y축을 기준으로
#    동, 북, 서, 남
dx = [0, -1, 0, 1]
dy = [1, 0, -1, 0]

# 현재 위치
x, y = 2, 2

for i in range(4):
    # 다음 위치
    nx = x + dx[i]
    ny = y + dy[i]
    print(nx, ny)
```
